Description of Service

Brevi is an AI-powered task brief engine for software development teams that operates as a Slack application. The Service includes:

  • Fetching task content from Jira Cloud and Shortcut via your provided API credentials
  • Scanning GitHub repository file structures to identify relevant files for a given task
  • Generating AI-powered structured briefs and summaries using third-party AI providers
  • Delivering task briefs to developers via Slack direct messages
  • Creating tickets in Jira or Shortcut directly from Slack via the /create-task command
  • Providing team management commands including /assign, /done, /my-tasks, /brevi-status, and /setup

The Service is provided on a subscription basis. A free tier is available with limited monthly task assignments. Paid plans unlock additional features and higher usage limits.
